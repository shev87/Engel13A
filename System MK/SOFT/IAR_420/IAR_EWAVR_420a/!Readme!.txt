1. Запустить id.bat.
2. Из полученного файла id.txt выудить ID (имеет вид шестнадцатеричного числа вроде 0x12345).
3. Вставить полученный ID в файле kg.bat вместо 0x12345.
4. Запустить отедактированный kg.bat (должен появиться файл Keys.txt).
5. Использовать содержимое файла Keys.txt для инсталляции нужного продукта (AVR, ARM, ...).
